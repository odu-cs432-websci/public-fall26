## Course Description

### Catalog Description

Provides an overview of the World Wide Web and associated decentralized information structures, focusing mainly on the computing aspects of the Web: how it works, how it is used, and how it can be analyzed. Students will examine a number of topics including web architecture, web characterization and analysis, web archiving, Web 2.0, social networks, collective intelligence, search engines, web mining, information diffusion on the Web, and the Semantic Web.

### Course Description

The Web has fundamentally changed how we learn, play, communicate, and work. Its influence has become so monumental that it has given birth to a new science: Web Science, or the science of decentralized information structures. Although Web Science is interdisciplinary by nature, this course will focus mainly on the computing aspects of the Web.

### Module 3: Measuring and Searching the Web

#### Module Overview

This module examines both the scale and structure of the Web and the computational mechanisms used to search it. Students explore how researchers estimate the size and growth of the Web, analyze its structural properties (including power law distributions), and study how search engines crawl, index, and rank webpages.

The module integrates theoretical concepts (web growth, link dynamics, precision/recall) with algorithmic techniques (TF-IDF and PageRank), preparing students to both measure and retrieve information from large-scale web collections

**Relevance**

The Web is vast, dynamic, and constantly evolving. Because it is impossible to measure it directly in its entirety, researchers rely on statistical and computational techniques to estimate its size and understand its structure.

At the same time, search engines provide practical access to this massive information space. Understanding how crawling, indexing, ranking, and evaluation metrics work reveals the computational foundations behind everyday web search.

### Module 5: Visualization and Analysis of Social Networks

#### Module Overview

This module explores social networks as structured and evolving systems. Students examine both offline and online social networks as graphs, where nodes represent individuals and edges represent social ties. The module introduces fundamental structural properties of networks, including strong and weak ties, triadic closure, tie strength, embeddedness, and how removing highly embedded edges can lead to graph partitioning.

Building on structural foundations, the module investigates how networks evolve over time through the mechanisms of selection and social influence. Students examine the principle of homophily—the tendency for individuals to associate with similar others—and analyze whether similarity in networks results from choosing like-minded connections (selection) or adopting behaviors from existing ties (influence).

Finally, students explore computational methods for visualizing social networks. The module introduces graph drawing and layout algorithms and demonstrates how to construct and visualize networks using tools such as Python libraries and JavaScript frameworks (including D3.js). Visualization techniques are used to analyze community structure and information flow within networks.

Together, this module integrates network theory, dynamic processes, and computational visualization to provide a comprehensive understanding of social networks in the Web context.

**Relevance**

Social scientists use network analysis to study how individuals interact and how social structures shape behavior. Online social networks (OSNs) provide unprecedented opportunities to examine these structures at scale. Understanding network topology, tie strength, and embeddedness allows researchers to better explain how communities form, fragment, and reorganize.

At the same time, the mechanisms of selection, homophily, and social influence are central to understanding how information, behaviors, and beliefs spread through networks. These principles are critical for studying phenomena such as the spread of inaccurate or misleading information, trust formation, and online polarization.

Visualization plays a key role in interpreting complex network structures. Effective network visualization helps reveal patterns of connectivity, community formation, and influence pathways that may not be apparent through numerical analysis alone.

Together, structural analysis, dynamic modeling, and visualization provide a rigorous framework for understanding how social networks function and evolve in the digital age
